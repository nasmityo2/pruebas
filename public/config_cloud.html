📋 Cómo Restaurar un Respaldo

  1. Descarga el archivo de respaldo deseado
  2. Cierra completamente BodegApp
  3. Renombra el archivo descargado a mi-tienda.db
  4. Reemplaza el archivo en la carpeta de datos de BodegApp
  5. Inicia BodegApp nuevamente
